# 📖 Project Overview

The **APL Logistics – Machine Learning-Based Late Delivery Risk Prediction System** is an end-to-end **Business Analytics and Machine Learning** solution developed as part of a **Business Analytics Internship Project**.

The objective of this project is to predict whether a shipment is likely to be delivered late **before dispatch**, enabling logistics managers to take proactive actions that reduce operational costs and improve customer satisfaction.

The project combines:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning
- Model Evaluation
- Business Intelligence Dashboard
- Predictive Analytics

into a single decision-support system built using **Python**, **Scikit-Learn**, and **Streamlit**.

---

> 🎓 **Internship Project**

This project was successfully completed as part of my **Business Analytics Internship at Unified Mentor Pvt. Ltd.**

During the internship, I designed and developed an end-to-end **Machine Learning-Based Late Delivery Risk Prediction System** for **APL Logistics (KWE Group)**. The project involved data preprocessing, exploratory data analysis (EDA), feature engineering, predictive modeling, model evaluation, and the development of an interactive Streamlit dashboard for business decision support.

# 🎯 Business Problem

Late deliveries are one of the biggest challenges in global supply chain operations.

Delivery delays can lead to:

- Increased transportation costs
- Customer dissatisfaction
- Revenue loss
- Poor resource utilization
- Inventory planning issues
- Reduced operational efficiency

Traditional logistics systems often identify delays only after they occur. This project addresses that limitation by using Machine Learning to predict high-risk shipments before dispatch, allowing logistics teams to take preventive action.

# 🤖 Machine Learning Models

The project evaluates multiple supervised Machine Learning algorithms to determine the most suitable model for predicting late delivery risk.

| Model | Purpose |
|--------|---------|
| Logistic Regression | Baseline Classification Model |
| Random Forest | Ensemble Learning |
| Gradient Boosting | Boosted Decision Trees |
| XGBoost | High Performance Gradient Boosting |

Each model was trained using the same preprocessing pipeline and compared using multiple evaluation metrics.

---

# 📈 Best Model Performance

The selected model achieved the following performance:

| Metric | Score |
|---------|-------|
| Accuracy | **73.27%** |
| Precision | **84.28%** |
| Recall | **62.99%** |
| F1 Score | **72.10%** |
| ROC-AUC | **0.8156** |
